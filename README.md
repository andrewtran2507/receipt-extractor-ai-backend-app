# Engineering Assessment Backend (NestJS)

## Description
```bash
An API for the Extract Receipt Website
```
Feature list handles
- **API - NestJS, TypeScript, @google/generative-ai, swagger, uuidv4**
  - receipt-extractor API
    - POST: /receipt-extractor/extract

## API

```bash
$ NestJS
$ TypeScript
$ @google/generative-ai
$ swagger
$ uuidv4
```

### Node v22.5.1

## Project setup

```bash
$ npm install
```
```bash
Crate the .env into the root of the project

GEMINI_API_KEY=<YOUR_GEMINI_API_KEY>
PORT=56789
SERVER_URL=http://localhost
```
```bash
change the <YOUR_GEMINI_API_KEY> yo your GEMINI API KEY
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
